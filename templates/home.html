CORGIS Immigration Data Set

The Dictionaries structure is as follows

The First Key is the Country and it's value is the countries name.

The next key is called Data, which has another dictionary for a value that possess the types of Immigrants that the US received from that country

For instance, theres a value called 'Legal Permanent Residents' which has another dictionary that details the types of legal residents.

Which are residents from last place they lived('Last Residence') or residents by birth('Birth').

The Dictinary finishes with a 'Year':'yyyy' value pair.

The rest of the json file runs through all countries that the US has received immigrants from.
